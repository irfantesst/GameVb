Deskripsi Game :
Text Twist ini adalah sebuah puzzle game yang di dalamnya kita bisa menebak kata tetapi tidak dalam bahasa Indonesia lagi melainkan bahasa Inggris.
Game ini bisa mengasah kemampuan grammar bahasa Inggris kita.

Cara Memainkan Game ini :
1. Untuk memulai permainan ini Pilih File--> New Game jika sudah berhasil maka terdapat 5 kotak yang berisi huruf
2. jika langkah pertama sudah maka kita wajib memilih 5 kotak yang berisi huruf tersebut sehingga membentuk suatu kata berbahasa inggris.
3. jika kita benar membentuk 1 kata berbahasa inggris maka kata tersebut akan masuk dalam list kata yang benar.
4. terdapat level di dalam game ini jadi setiap naik level maka kata yang di bentuk akan semakin susah.
